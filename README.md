# spring-boot-smpp
Spring boot starter for SMPP

## Installation

```xml
    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>
```

```xml

    <dependency>
        <groupId>biz.asplund</groupId>
        <artifactId>spring-boot-smpp</artifactId>
        <version>0.0.1-SNAPSHOT</version>
    </dependency>
```
